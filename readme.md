# is-scoped

> Check if a string is a [scoped npm package name](https://docs.npmjs.com/misc/scope)

## Install

```
$ npm install is-scoped
```

## Usage

```js
import isScoped from 'is-scoped';

isScoped('@sindresorhus/df');
//=> true

isScoped('cat-names');
//=> false
```
